Minimal Goal

Convert a single Office document to PDF using:

  • App-only authentication
  • One SharePoint site
  • One document library
  • Temporary working folder
Required Settings
  • TenantId
  • ClientId
  • ClientSecret
  • SiteUrl
  • LibraryName
  • TempFolder
Setup Walkthrough
Loops automatically. Does not open in lightbox.
Definition of Done
  • Input file uploaded
  • PDF downloaded successfully
  • Temp file cleaned up
  • No manual SharePoint steps required

Minimal PDF Conversion Flow

This is the smallest viable slice of GraphLib: upload → convert → download → delete.

sequenceDiagram autonumber actor Caller participant Runner as GraphLibPdfRunner participant MSAL participant Graph participant SharePoint Caller->>Runner: RunAsync("input.docx") Runner->>MSAL: AcquireTokenForClient() MSAL-->>Runner: access_token Runner->>Graph: Resolve Site + Drive Runner->>SharePoint: Upload file to temp folder Runner->>Graph: Download as PDF (?format=pdf) Graph-->>Runner: PDF stream Runner->>SharePoint: Delete temp file Runner-->>Caller: Success (output.pdf)
Key Principle: SharePoint is treated as a temporary rendering engine — not as long-term storage.
Minimal Console Usage
dotnet run --project GraphLib.Console `
  --file "C:\Docs\Alpha.docx"
Minimal Settings Example
{
  "TenantId": "YOUR-TENANT-ID",
  "ClientId": "YOUR-CLIENT-ID",
  "ClientSecret": "YOUR-CLIENT-SECRET",
  "SiteUrl": "https://contoso.sharepoint.com/sites/MySite",
  "LibraryName": "Documents",
  "TempFolder": "_graphlib-temp",
  "ConflictBehavior": "replace"
}
Error Handling (Minimal Version)
  • 403 → Permission scope issue (Sites.Selected not granted)
  • 404 → Site or library mismatch
  • 401 → Token acquisition failure
